Tree of Knowledge (tok)

Tree of Knowledge (tok) is a command line tool that I made and use to manage my notes. tok organizes a document so that it may be read sequentially - no jumping back and forth to figure out what the author is referring to. tok creates a tree, where each node is a YAML file containing LaTeX formatted text and dependency information, and uses this dependency information to sort the nodes in such a way that the text contained in YAML files may be read/studied sequentially. tok outputs a TEX file and runs XeLaTeX to generate a PDF. tok scales to arbitrarily large trees, leading to generating arbitrarily large documents, such as textbooks. I use tok to write papers and internal reports/documentation. I also use it to generate to-do lists when I'm not sure what to do first, especially with a big project.
